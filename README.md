# Image-Recognition
Image recognition project using Teachable Machine by Google.

In this project an Image recognition model was trained using Teachable Machine by Google to recognize different image classes.

### Teachable Machine
Using the Teachable Machine website the image recognition model was trained to recognize lions, tigers and cheetahs.
Four images were uploaded to each class. After training the model the python Tensorflow version is exported and 
downloaded to the device, also the python code is copied and will be used later.

### Anaconda
Anaconda software is used to create a virtual environment, which is an isolated python installation. Virtual environments
are used so that different versions of python and packages can be used for different projects.

### Visual Studio Code
The IDE used for this project is Virtual Studio code. The two files from the trained model is added to a folder along
with the python code that was copied earlier from the Teachable Machine website, also a test image
is added and its path is refrenced in the code to examine the trained model recognition capabilities.

### Results
The trained image recognition model successfully recognised the test image with a high confidence score.
